# time-perception

**Purpose**: Surreal immersive site about time perception — domestic horror aesthetic. A room you've been in too long.

**Status**: Active

**Tech**: Single-file HTML. Victorian SVG wallpaper (canvas, breathing animation), Web Audio API ticking (generated, scroll-speed-synced), SVG clocks with scroll-driven distortion, stuttered section reveals, session timer.

**URL**: https://asafagranatprosus.github.io/pages/apps/time-perception/

## Concept

Time as intimate and wrong — not cosmic, not Dali. The way a clock sounds different at 3am. The way a childhood room feels smaller when you return. Five sections: Entry → The Room → The Clocks → The Receipt → The Exit.

## Features

- Breathing Victorian wallpaper (canvas, parallax with scroll)
- Web Audio ticking that speeds up with scroll velocity
- 4 SVG clocks showing wrong times, distorting on scroll
- Receipt that prints line by line as you scroll into the section
- Session duration counter (how long you've been in the room)
- Stuttered memory-like section reveals
