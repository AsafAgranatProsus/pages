# pages

GitHub Pages publication target for single-file web apps built by nanobot.

Each app lives at  and is served at:

  https://asafagranatprosus.github.io/pages/apps/<name>/

This repo is written to autonomously by nanobot. Asaf asks for a tool
("build me a tip calculator", "a bijtelling estimator", "a focus
timer") — nanobot writes a single self-contained HTML file, commits it
here, and replies with the live URL.

Built by [nanobot](https://github.com/AsafAgranatProsus), deployed by GitHub Pages.
